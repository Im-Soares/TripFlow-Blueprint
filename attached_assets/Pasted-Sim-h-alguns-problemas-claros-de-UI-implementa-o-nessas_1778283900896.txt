# 1. Ícones não aparecem

## Problema

Os ícones estão a renderizar como:

* quadrados
* caracteres estranhos
* símbolos quebrados

Isso normalmente acontece por uma destas razões:

### Possíveis causas

#### A) Font package não carregado

Muito provável.

Se estiveres a usar:

* Lucide
* FontAwesome
* Material Icons
* Ionicons
* Heroicons

a font pode não estar importada corretamente.

---

#### B) Nome dos ícones inválido

Exemplo:

```tsx
<HomeIcon />
```

mas o package espera:

```tsx
<Home />
```

---

#### C) Ícones com cor igual ao fundo

Menos provável aqui, porque aparecem “símbolos estranhos”.

---

#### D) Font family quebrada

Muito provável se:

* estás a usar Expo
* React Native
* custom fonts
* Tailwind Native

---

# O que eu faria imediatamente

## Se for React Native + Expo

Verificar:

```bash
npx expo install @expo/vector-icons
```

Depois:

```tsx
import { Ionicons } from '@expo/vector-icons';
```

E:

```tsx
<Ionicons name="home-outline" size={24} color="#8B5CF6" />
```

---

## Se for Lucide React Native

Provavelmente o problema.

Alguns ícones não funcionam bem em RN dependendo da versão.

Melhor solução:

```bash
npm install lucide-react-native
```

Depois:

```tsx
import { Home, Map, User } from "lucide-react-native";
```

# 2. Botões cortados em “My Trips”

## Problema

Os category pills:

* têm altura demasiado pequena
* padding vertical insuficiente
* font line-height errada

Resultado:

* texto cortado
* labels invisíveis

---

# Causa provável

Tens algo tipo:

```tsx
height: 28
```

ou:

```tsx
py-1
```

quando devias ter:

```tsx
minHeight: 36-40
paddingVertical: 8
```

---

# Como resolver

## Pills corretas

```tsx
<View className="px-4 py-2 rounded-full bg-[#151525] mr-2">
  <Text className="text-sm text-white">All</Text>
</View>
```

ou:

```tsx
height: 40
paddingHorizontal: 16
justifyContent: center
```

# 4. Ícones alinhados mal nas settings

Na página profile:

* os ícones da esquerda estão esmagados
* parecem 12x12 dentro de caixas grandes

## Causa

Container maior que o icon size.

Provavelmente:

```tsx
w-10 h-10
```

mas:

```tsx
size={12}
```

---

# Correção ideal

```tsx
<Settings size={18} color="#8B5CF6" />
```

Container:

```tsx
w-10 h-10 rounded-xl items-center justify-center
```
